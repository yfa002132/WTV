FAQ

常见问题

两者共享内容源与账号体系，但 UI 与交互针对不同设备优化：手机版为竖屏 + 底部导航；电视版为 Leanback 大屏布局，支持遥控器焦点导航，并额外提供精选页、排行榜等入口。包名分别为 com.film.wtv 与 com.film.wtv.tv，需分别安装。
macOS：下载对应芯片架构的 .dmg 文件，打开后将 WTV 拖入「应用程序」即可。Apple Silicon 请选择 arm64 版本，Intel Mac 请选择 x64 版本。

Windows：运行 .exe 安装程序按向导完成安装；便携版解压 zip 后直接运行即可。

Linux：AppImage 需先执行 chmod +x 赋予可执行权限后双击运行；deb 包可通过 sudo dpkg -i 安装。
可通过 U 盘拷贝 APK 至电视安装，或使用 adb install 命令。若 PatchWall 提示不兼容，请下载 armeabi-v7a 版本。安装后在应用列表或 Leanback 启动器中找到 WTV 即可。
手机端在播放页或详情页点击投屏按钮，选择同一局域网内的 DLNA 设备即可推送。电视端可在个人中心开启「投屏接收端」服务，接收来自手机的投屏内容。
应用内「个人中心 → 意见反馈」可提交问题。也可通过 QQ：416528206 或邮箱 163@outlook.com 联系我们，或在 GitHub 项目 WTVPlayForAndroid 提交 Issue。
